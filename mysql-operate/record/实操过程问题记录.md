**Demo01:对表、数据操作(DDL,DML)**


> 问题:向表插入含中文数据时连续报错(ERROR 1366：中文编码字符串非法)
> 原因：建表未指定`utf8mb4`字符集，表默认字符集为`latin1`，不兼容中文 UTF-8 编码；  
> 解决：建表时直接指定字符集。
> ```
> CREATE TABLE user_table ( 
> id INT(11), 
> name VARCHAR(50), 
> age INT(11),
> gender VARCHAR(1) 
> ) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COLLATE utf8mb4_unicode_ci;
